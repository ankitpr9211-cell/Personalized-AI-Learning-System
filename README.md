# Personalized AI Learning System

An AI-powered Study Assistant built using Streamlit, LangChain, Groq LLM and ChromaDB.

## Features

- Ask AI
- Chat with PDF (RAG)
- Quiz Generator
- AI Study Planner
- Daily Study Streak
- Multiple PDF Support
- Beautiful Animated UI

## Tech Stack

- Python
- Streamlit
- LangChain
- Groq
- ChromaDB
- HuggingFace Embeddings

## Installation

pip install -r requirements.txt

streamlit run app.py
